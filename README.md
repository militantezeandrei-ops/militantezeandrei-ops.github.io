# ZDM_ | Zeandrei D. Militante Portfolio 

![Zeandrei Militante Portfolio Showcase](https://raw.githubusercontent.com/militantezeandrei-ops/militantezeandrei-ops.github.io/main/assets/profile.jpg)

A high-performance, cyberpunk-inspired professional portfolio built for **Zeandrei D. Militante**, a Web & App Developer from Marilao, Bulacan. This project showcases technical expertise through a futurist aesthetic, utilizing modern web technologies and a precision-focused design system.

---

##  Concept: The Digital Workspace
The core concept behind this portfolio is the **"Terminal/Nexus"** aesthetic. It aims to bridge the gap between raw code and polished user experience, reflecting the developer's dual expertise in backend logic and frontend design.

- **Futurist Aesthetic**: Inspired by cyberpunk and high-tech command interfaces.
- **Precision Focus**: Clean layouts with technical details (mono fonts, coordinates, and scanlines).
- **Interactive Feedback**: A "living" UI that reacts to mouse movements and scroll positions.

---

##  Design System

### Visual Language
- **Neon Glow**: Strategic use of `#00FF41` as the primary action color to guide the user's eye.
- **Grid Patterns & Scanlines**: Subtle background textures that evoke a digital matrix/CRT screen feel.
- **Glassmorphism**: Semi-transparent "glass" cards with backdrop blurs to add depth in a dark environment.
- **Corner Brackets**: Coding-inspired UI elements used to frame important sections.

### Color Palette
| Color | Hex | Role |
| :--- | :--- | :--- |
| **Neon Green** | `#00FF41` | Primary Accent, Glows, Actions |
| **Deep Black** | `#000000` | Main Background |
| **Dark Nexus** | `#0a0a0a` | Section Backgrounds |
| **Terminal Card** | `#0f0f0f` | Functional Cards |
| **Carbon Border** | `#1a1a1a` | UI Borders |
| **Cyber Light** | `#e0e0e0` | Text Content |

### Typography
*   **Headings & Display**: `Rajdhani` (Google Fonts) — A geometric, technical sans-serif that fits the high-tech theme.
*   **System & Mono**: `Share Tech Mono` (Google Fonts) — For technical labels, code snippets, and terminal-style information.

---

##  Tech Stack

- **Frontend**: TailwindCSS (Custom configuration), Vanilla CSS (Complex animations).
- **Library**: Lucide Icons (Iconography).
- **Architecture**: Single Page Application (SPA) structure with modular design.
- **Logic**: Vanilla JavaScript for scroll reveals, cursor glow, and typewriter effects.
- **Performance**: Optimized for fast load times and clean mobile responsiveness.

---

##  Key Features

- **Cursor Glow**: A dynamic light source that follows the user's mouse, illuminating the "grid" background.
- **Scroll Reveal**: Smooth, high-performance entrance animations for content blocks.
- **Typewriter Effect**: Dynamic header changes to highlight different roles/skills.
- **Project Showcase**: Detailed card system with hover-activated previews.
- **Mobile Optimized**: Fully responsive navigation and layout for all device sizes.

##  Installation

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/militantezeandrei-ops/militantezeandrei-ops.github.io.git
    ```
2.  **Open the directory**
    ```bash
    cd militantezeandrei-ops.github.io
    ```
3.  **Launch**
    Open `index.html` in any modern web browser or use a Live Server extension (recommended).
